## Installation and Set Up

1. Install dependencies

```sh
npm install
```

2. Start the development server

```sh
npm run dev
```

## Building and Running for Production

1. Build the application for production usage

```sh
npm run build
```

2. Start a production server

```sh
npm run start
```
